## 题目分析：

问：小于等于n的非负整数中共出现了多少次‘1’？

### 解题思路（1）
暴力枚举所有小于等于n的非负整数，统计‘1’出现的次数。 时间复杂度是‘ O(n) ’的。
### 解题思路（2）
换一种统计方式，改为统计各个数位上1出现的次数。比如，先统计个位1出现的次数，再统计十位1出现的次数，以此类推，时间复杂度是‘ O(logn) ’的。
#### 算法的正确性：
要保证正确统计各个数位1出现的次数，需要分类讨论。如果当前位是0，那么前缀的数必须要更小，而后缀数任意；如果当前数是1，前缀的数更小时后缀数任意，前缀的数相同时后缀数要更小；如果当前数大于1，则前缀的数要小于等于，后缀的数任意。
以1013为例。个位1出现的次数是102（前缀000~101），十位1出现的次数是10（前缀00~09）*10（后缀任意）+4（前缀10，后缀0~3） = 104， 百位1出现的次数是1（前缀0）*100（后缀任意）= 100，千位1出现的次数是14（后缀000~013）= 14，共320。
